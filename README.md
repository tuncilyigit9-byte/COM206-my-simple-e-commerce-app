# Shopping App

This project is a simple shopping application developed using Flutter. Users can browse products by categories, customize items by selecting color and size, easily access their cart from multiple screens, and complete their purchases.

## Features

- **Category Selection**: Choose from Hat, T-Shirt, Shirt, Jeans, and Coat categories
- **Product Customization**: Select color and size for each product
- **Cart Management**: Add products to cart, view total price, and quickly navigate to the cart from the category or product pages
- **Responsive Design**: Optimized interface for mobile devices

## Installation

Follow these steps to run this project:

### Requirements

- Flutter SDK (3.11.4 or higher)
- Dart SDK
- Android Studio or Visual Studio Code (with Flutter extension)

### Steps

1. Install and set up Flutter:
   ```bash
   flutter doctor
   ```

2. Clone or download the project:
   ```bash
   git clone https://github.com/tuncilyigit9-byte/COM206-my-simple-e-commerce-app.git
   cd COM206-my-simple-e-commerce-app
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the application:
   ```bash
   flutter run
   ```

## Usage

1. Click the "Start Shopping" button from the home page
2. Select your desired category
3. Choose color and size for the product
4. Add the product to cart with "Add to Cart"
5. Navigate to your cart using the "Go to Cart" button on the Category or Product pages
6. View selected products and the total price on the Cart page
7. Complete the order by clicking the "Pay" button
